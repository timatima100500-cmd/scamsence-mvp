## Технические заметки

### Bash / Windows mount sync
Файлы написанные через Cowork Write tool видны в bash с задержкой (NTFS mount caching).
**Решение:** писать через `python3 - << 'PYEOF'` напрямую в bash — тогда sync мгновенный.

### Git lock files
`.git/index.lock` и `.git/config.lock` — оставались после bash sandbox операций.
**Решение PowerShell:**
```powershell
Remove-Item ".git\index.lock" -Force
Remove-Item ".git\config.lock" -Force -ErrorAction SilentlyContinue
```

### GitHub auth (на будущее)
- Токен типа `github_pat_` (fine-grained) даёт 403 — нужно явно разрешать репо
- Использовать **classic token** (`ghp_`) с галочкой `repo`
- Создать: https://github.com/settings/tokens/new
- Вставить в URL: `https://USERNAME:TOKEN@github.com/...`
- После push сбросить URL обратно (без токена)
