   ```powershell

   ```
   (Если спросит — ответить `yes`)

2. **Создать пустой репо на GitHub:** https://github.com/new → `scamsence-mvp` → без README/gitignore

3. **Подключить remote и push:**
   ```powershell
   cd "C:\Users\Acer\Documents\Claude\Projects\ScamSence_MVP"
   git remote add origin git@github.com:timatima100500-cmd/scamsence-mvp.git
   git push -u origin main
   ```

4. **Если git status даёт ошибки** (lock файлы) — удалить:
   ```powershell
   Remove-Item .git\config.lock -ErrorAction SilentlyContinue
   Remove-Item .git\index.lock -ErrorAction SilentlyContinue
   ```
